ifconfig ( ifconfig sudo ifconfig eth0 192.168.1.10 netmask 255.255.255.0 up ) :- to display the confogurations of network 


traceroute (traceroute google.com) :- to display the route and measure transit delays of packets across an internet protocol network 

mtr ( mtr google.com, mtr 8.8.8.8) :- mtr is used to combine the functionalities of the ping an traceroute commands

iwconfig ( iwconfig, sudo iwconfig wlan0 essid "MyWiFi") :- is typically used to configure wireless network interfaces on Linux systems
